# Logistic Regression Classification – Social Network Ads

A machine learning project that uses **Logistic Regression** to predict whether a user will purchase a product based on **Age** and **Estimated Salary** from a social network advertisement dataset.

This repository contains a complete beginner-friendly classification workflow in **Python** using **Jupyter Notebook**, including data loading, exploration, visualization, and model building.

## Overview

Online advertising platforms often want to predict whether a user will purchase a product after viewing an advertisement.

This project applies **Logistic Regression**, a supervised machine learning algorithm for binary classification, to predict whether a user will make a purchase based on:

- **Age**
- **Estimated Salary**

The notebook demonstrates the end-to-end process of solving a classification problem using Python.

---

## Objectives

The main objectives of this project are to:

- Load and inspect the dataset
- Understand the structure of the data
- Perform exploratory data analysis
- Visualize feature distributions and relationships
- Build a Logistic Regression classification model
- Interpret the model output
- Understand how demographic features affect purchase behavior
